For scanning commands:

    Ip Port scanning:
        scan (ip or site domain) (starting port) (ending port)
        Example command:
            scan 0.0.0.0 20 90

    Admin panel scanning:
        pscan (site url) (Max thread is optional)

        Example command:
            pscan https://example.com


    Domain scanning:
        dom (site name) (site protocol)

            Example command:
                dom youtube https
    Banner lookup:
        bn (ip) (port)
        Example command:
            bn 0.0.0.0 22
